# Банковский виджет

## Виджет выполняет следующие операции:

1. Маскирует номера карт и счетов
+ описание работы 
```
- на вход подается число - 7158300734726758
- резульатом функция возращает - 7158 30** **** 6758 
```
2. Фильтрует список с информацией карт по заданному параметру
+ описание работы 
```
- на вход подается список с информацие по нескольким картам и параметр state
- на выходе отфильтрованный список с информацией о картах, значение state которых равно заданному. 
```

3. Сортирует список с информацией карт по дате
+ описание работы 
```
- на вход подается список с информацие по нескольким картам и направление сортировки
- на выходе отсортированный по даам список
```

4. Фильтрует списки с описанием транзакций, хранящихся в словарях и возвращает итератор со словарями
+ описание работы 
```
- на вход получаем список транзакций и наименование валюты по которой мы хотим отсортировать его
- на выходе возвращает итератор с отфильтрованным списком
```

5. Генерирует описание транзакций, взятое из списка этих транзакций
+ описание работы 
```
- на вход принимает список словарей с транзакциями
- возвращает генератор с описаним каждой операции
```
6. Генерирует номера карт в заданном диапазоне
+ описание работы 
```
- на вход получает начальное и конечное значения
- возвращает генератор номеров карт в заданном диапазоне
```
7. Конвертирует валюту 
+ описание работы 
```
- получает на вход словарь с данными о транзакции.
- конвертирует валюту из USD и EUR в RUB и возвращает сумму транзакции в рублях
```
8. Чтение jsonn файлов 
+ описание работы 
```
- принимает путь к json-файлу в качестве аргумента.
- возвращает список словарей с данными о финансовых транзакциях или, если
  файл пустой, содержит не-список или не найден, возвращается пустой список.
```
9. Чтение csv и xlsx файлов 
+ описание работы 
```
- принимает путь к csv или xlsx файлам в качестве аргумента.
- возвращает список словарей с транзакциями
```

## Декораторы

### Модуль decorators.py содержит следующие декораторы:

    1. decorator_with_args

    Логирует работу модулей. Может записывать данные в файл, либо выводить в кончсоль. 
    В файл записывается время запуска, в случае ошибки, записывает текст ошибки и 
    аргументы переданные функции, если ошибки не возникло, записывает информацию об
    успешном завершении.
