# Банковский виджет

## Виджет выполняет следующие операции:

    1.Маскирует номера карт и счетов

    2.Фильтрует список с информацией карт по заданному параметру

    3.Сортирует список с информацией карт по дате

## Модули для тестирования

    1. Модуль test_masks для тестирования модуля src/masks.py
    
    2. Модуль test_widget для тестирования модуля src/widget.py

    3. Модуль test_processing для тестирования модуля src/processing.py
